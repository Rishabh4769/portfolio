# Rishabh Portfolio

React portfolio app built with Vite.

## Run

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`

## Structure

- `index.html` - Vite HTML entry
- `src/main.jsx` - React bootstrap
- `src/App.jsx` - portfolio page content
- `src/styles.css` - visual system and layout
